# ✈️ Aviation Planner

Flight planning system for general aviation pilots.

## Problem

General aviation pilots need to consult multiple sources to plan a flight:
distance, estimated time, fuel consumption, weather, NOTAMs and alternates.

This project demonstrates the ability to build an aeronautical planning system
following modern software engineering practices.

## MVP

| Input        | Output             |
|--------------|--------------------|
| Origin ICAO  | Distance (NM)      |
| Destination  | Flight time        |
| Aircraft     | Fuel estimate      |
|              | Route on map       |
